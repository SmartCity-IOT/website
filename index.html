IoT Security Lab

project focuses on Noise, CO₂, and Ultrasound sensors to demonstrate how IoT supports sustainable and smarter city management

Project Purpose

Smart Cities use Internet of Things (IoT) technologies to improve urban living. By deploying sensors across the city, governments can monitor environment, traffic, and resources in real-time.

IoT Security Analysis

Comprehensive analysis of IoT device vulnerabilities, network protocols, and communication security in controlled lab environments.

Attack Simulation

Ethical simulation of common cybersecurity attacks including DDoS, MITM, and Brute force , SQL injection sniffing, for educational purposes.

Dataset Generation

helps in: Reducing pollution (air & noise) Managing resources efficiently (waste, energy, parking) Improving citizen health and safety.
